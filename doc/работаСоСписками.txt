Функции по работе с базами для клиента:

1. Импортировать список номеров из Эксель, txt, буфера
2. Добавлять в уже имеющийся в кабинете список отдельные номера
3. Удалять выделенные номера из любого списка в личном кабинете
4. Давать списку номеров имя
5. Переименовывать список
6. Соединять два разных списка в один с присвоением нового имени
7. При создании рассылки выбирать несколько списков выбирая чекбоксы
Поиск конкретного номера в списке. (Если клиент хочет отправить тестовое сообщение на конкретный номер из списка)
